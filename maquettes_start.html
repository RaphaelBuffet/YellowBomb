Maquette visuelle de votre projet

Par Jean Dupont le 5/02/2019 à 09:00

Inspiré du célèbre jeu-vidéo Bomberman, notre jeu a pour thème les gilets jaunes.
Le premier monde est tel quel que l'on peut le trouver dans les jeux-vidéos, les seconds ont été créés par nous-mêmes.
Le personnage principal, lui, est un gilet jaune punk. Nous avons décidé de le représenter ainsi car c'est un casseur, donc un "méchant". Hors, dans les années où Bomberman est sorti, les punks étaient perçus dans l'imaginaire collectifs comme des crapules.
